# VIZIIA Studio — AI Photoshoot Platform

## Project
React + Vite + TypeScript + Tailwind CSS
Live at: https://demo.viziiastudio.com
GitHub: https://github.com/viziiastudio/viziia-app

## Stack
- React 18, TypeScript, Tailwind CSS
- Vite build tool
- Deployed on Netlify

## Deploy
npm run build && netlify deploy --prod --dir=dist

## Key Files
- src/components/steps/ — 5-step wizard
- src/components/Sidebar.tsx — config panel
- src/components/BottomNav.tsx — navigation

## Business Context
AI-driven photography studio for eyewear brands.
Produces campaign visuals faster and cheaper than traditional photoshoots.
